# E-commerce Frontend Clone

This project is a frontend e-commerce application built using **Next.js and React**


## Live Demo

---

## GitHub Repository

https://github.com/Shivam-Tidke/bharatGo-e-commerce

---

##  Tech Stack

- Next.js (React)
- Tailwind CSS
- Platzi Fake Store API

##  Features

- Product listing from API
- Add to cart functionality
- Increase / decrease product quantity
- Orders page
- Orders stored in localStorage
- Responsive UI 
---

##  Application Flow

1. Products are fetched from the Platzi Fake API
2. Products can be added to the cart
3. Cart opens from the topbar
4. Users can update quantity or remove items
7. On checkout, cart items are saved as orders
